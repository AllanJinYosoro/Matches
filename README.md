# Matches

Matches 是火柴 2.1.0.11 的独立后继项目。当前版本复现双击 Ctrl 唤出、本地文件搜索和可自定义快捷入口，不包含账号、聊天及在线服务。

## 构建

在仓库目录打开 PowerShell：

```powershell
.\build.ps1
```

脚本会从 voidtools 官方地址下载固定版本的 Everything 与 ES，校验 SHA-256，编译 `dist\Matches.exe`，并运行内置检查。

## 打开与验收

1. 双击 `dist\Matches.exe`。首次运行时接受 UAC，以安装 Everything 官方搜索服务。
2. 未输入搜索词时，窗口显示预设快捷入口；单击即可启动。
3. 单击右下角“添加”，可加入文件、程序或文件夹；右键快捷入口可以删除，设置菜单可以恢复默认。
4. 输入本机文件或文件夹名称，确认快捷入口切换为搜索结果。
5. 用方向键选择结果并按 Enter，或双击打开；右键可定位文件和复制完整路径。
6. 快速按两次 Ctrl 隐藏窗口，再按两次 Ctrl 重新唤出。Escape 只清空当前搜索词。
7. 从托盘图标或右下角设置菜单选择“退出 Matches”，确认程序结束。

设置菜单中的“开机自动启动”使用当前用户的 Windows 启动项；启用后登录时会在托盘后台启动 Matches。

Everything 索引、配置和快捷入口只写入 `%LOCALAPPDATA%\Matches`，不会读取或修改火柴安装目录中的数据库、配置和用户数据。
